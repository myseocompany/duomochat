# Instrucciones del repositorio

- La version visible del sistema se muestra en el footer desde `/Users/projects/duomochat/resources/views/layouts/footer.blade.php`, pero el valor fuente se cambia en `/Users/projects/duomochat/VERSION`.
- La fecha visible de ultima actualizacion del footer se cambia en `/Users/projects/duomochat/LAST_UPDATED`.
- Cada vez que hagas cambios en este repositorio, actualiza `/Users/projects/duomochat/VERSION` y `/Users/projects/duomochat/LAST_UPDATED` antes de terminar la tarea para que el footer refleje la nueva version y la fecha correcta.
