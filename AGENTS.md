# Instrucciones del repositorio

- La version visible del sistema se muestra en el footer desde `/Users/projects/duomochat/resources/views/layouts/footer.blade.php`, pero el valor fuente se cambia en `/Users/projects/duomochat/VERSION`.
- Cada vez que hagas cambios en este repositorio, actualiza `/Users/projects/duomochat/VERSION` antes de terminar la tarea para que el footer refleje la nueva version.
